# 🐄 Qurbanir-Hat(Premium Livestock Booking Platform)

## 📖 Project Purpose
QurbaniHat is a modern, SaaS-level digital marketplace designed to simplify the process of purchasing livestock for Qurbani. Users can seamlessly browse a curated collection of healthy cows and goats, view comprehensive animal details (breed, weight, age, pricing), and securely place a booking. Built with a focus on trust, transparency, and high-end UI/UX, the platform connects buyers with premium livestock delivered safely to their doorstep.

## ✨ Key Features
* **Ultra-SaaS UI/UX:** A premium, fully responsive design featuring modern glassmorphism, smooth hover effects, and a cohesive Emerald/Teal color palette.
* **100% Mobile Responsive:** Flawless experience across mobile phones, tablets, and desktop displays.
* **Secure Authentication:** Robust login and registration system powered by **Better Auth** (Email/Password & Google Social Login).
* **Protected Routes:** Middleware ensures that only authenticated users can access the Animal Details and My Profile pages.
* **Dynamic Animal Marketplace:** Browse from a JSON data source of 20 premium livestock (10 Cows, 10 Goats) with dynamic routing for individual details.
* **Simulated Booking System:** Users can fill out a delivery form to book an animal, complete with instant Toast Notifications upon success.
* **User Dashboard & Profile:** Private profile page where users can view and update their personal information (Name and Profile Photo).
* **Smooth Animations:** Integrated `animate.css` for professional, non-intrusive page-load animations.

## 📦 NPM Packages Used
* **[Next.js (App Router)](https://nextjs.org/):** Core React framework for SSR and routing.
* **[Tailwind CSS](https://tailwindcss.com/):** Utility-first styling framework used exclusively for the UI (No DaisyUI).
* **[Better Auth](https://better-auth.com/):** Comprehensive, secure authentication handling.
* **[Mongoose](https://mongoosejs.com/):** MongoDB object modeling to store user data securely.
* **[Animate.css](https://animate.style/):** Used to fulfill the assignment requirement for third-party animation libraries, bringing the SaaS design to life.
